Akkodis - TurtleBot3 Work

This project involved enhancing a TurtleBot3 service robot’s autonomy by integrating a RealSense D405 depth camera for 3D vision and a YOLO model for real-time object detection. A distributed ROS 2 system was used: an onboard Raspberry Pi 5 ran “ROS2 Jazzy,” while a workstation ran ROS 2 Humble Hawksbill. A custom ZeroMQ bridge transferred camera data and YOLO results between Pi and PC, with Eclipse Cyclone DDS improving reliability and reducing latency. The final system let the robot identify and fetch objects in a test environment.
